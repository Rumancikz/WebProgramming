1. Animation is a method in which figures are manipulated to appear as moving images.

Pros to animation are that it can bring life to the animation, gain a viewers attention, Can be fun to watch, and educational games and quizzes allow the visitor to interact and learn.

Cons to animation are that it takes alot of effor to create even a basic animation, needs skill in using the animation software such as Flash, Too much animation can be distracting and even annoying , can take up a lot of bandwidth, so will take so long to show over a slow internet connection, and needs the correct addin view the file format.

2. The benefits to using jQuery for animating is that it is simple and only takes a little bit of programming knowledge to create pleasing to look at animations. It is also super flexible because it allows user to add plug-ins. The limitations to jQuery is that it is an open source and not a set standard amonmg providers so the developers do not have te money, time or ability to fix issues. You may never find solutions to some problems. Along with that it is much slower then CSS, since its simple it is easy to pick up but it is a curse because it is not meant for client side interactions. If you misuse jQuery the code can get out of hand repeating itself over and over again. The community is trying to fix it, but it is a big problem.
